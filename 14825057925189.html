Gitop 設計分享

http://Gitop.io 的設計是我為一個朋友做的。

Gitop

Gitop 是什麼？某日，D 先生和我說，做了一個小工具，來爬 Github 的項目，並且按照一定排序。D 先生和我說，他每天都可以在上面看到很多很好的項目。

幾日後，D 先生在早上把我吵醒，他想起一個名字。

我們討論過 GitRank, GitPR,Git... 我已經想不起那些沒定下來的名字了，其實有很多不錯的，可惜就是註冊不上。

我突然想到，Git top, Gitop。我 check 了一下，Gitop 可用。就告知，這樣，我們就把名字定下來了。

http://gitop.io ﹣ Gitop.io

設計

我沒有怎麼參與設計，我只是提過一些意見。

但是，Github 登入需要一個圖片。

我開始聯想一個 Github 那種技術開源項目要用這個什麼樣的樣子。我沒有想到什麼答案。

我在某日，隨意瀏覽 App Store 時候，我看到一個 Atom（原子），所以，我確定了以原子作為元素來設計。

你們會看到圖標版本和圖片版本的 Atom 圖案有所出入，是因為我調整了線條粗細之後的結果，太細的線條，在圖標上壓根就看不清了。

END

就這樣就發了，他挺高興的。

你們可以去看看他的網站 Gitop.io
