MWeb ＋ 自定義模板 = 個人或企業網站

  • Written by Qoli Wong
  • Posted in 服務
  • 2016/12/23 18:36 下午

樣本工作室的全新服務項目。

MWeb 是一款本地 Markdown 工具，具備卓越的性能，出色的用戶體驗。同時具備良好的拓展性。所以樣本工作室基於 MWeb 的優越性，計劃開展此服務項目。

樣本工作室「全新服務」
基於 MWeb 作為後台系統
設計并開發其靜態化模板
純靜態化帶來前所未有的低廉價格和卓越性能

服務特色

由於 MWeb 具備出色的用戶體驗，以及 Markdown 也是一款易於學習的語言。

特色：
  • 基於 MWeb 作為網站後台設計。
  • 易於學習、隨處可用的 Markdown 語言。
  • 本地預覽、本地調整，再上傳線上。
  • 本地編寫，後台以 MWeb 程式為中心，跟隨 MWeb 發展。可體驗到流暢同步沒有卡頓的「原生後台體驗」

適合群體

適合于不想學習線上後台系統。如同記事本一樣的本地工作、還有 iOS 版本的 MWeb 原生體驗的用戶群體。

網站可用於：
  • 各種品牌官網
  • 攝影師、設計師的線上作品集
  • 文字工作者的線上網站
  • 個人知識、文章集線上網站

純靜態化網站的好處

分析一個東西的好壞，我們可以基於成本以及優點來分析。

託管一個純靜態的網站成本很低，如果使用七牛 CDN 這類服務來託管，一個月就有免費的 10G 流量使用。基本上就足夠一些小型實體店的需要了。
