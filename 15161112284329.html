這會是現在最合適 iPhone X 設計的壁紙 App

  • Written by Qoli Wong
  • Posted in 作品
  • 2018/1/16 22:0 下午

iPhone X 採用了 OLED 屏幕，如果使用全黑的壁紙將會更省電。
如果不喜歡全黑的壁紙，採用基於綠色系列的壁紙也能減少壽命損耗。但是千萬不要使用太多藍色的壁紙，藍色顯示單元格壽命比較短，所以不要用太多哦。

因此，我走上了一條尋覓壁紙的道路了。但是在尋覓大約 8 個壁紙 App 之後，我選擇了另一條路。

設計和開發

當然，我們無法逃脫先用 Sketch 確定大概的視覺風格，再開工開發的固定流程。
這是我第二次用 Swift 寫 App 了，所以用起來就感覺適應很多。
而且最大的好處是，Swift 怎麼寫都有代碼提示，不像以前在寫 JavaScript 一樣，代碼都是依賴腦子記憶。

再搭配上 Fastlane 發佈，我已經忘記了怎麼透過 xcode 發佈 App 了。


可能是最合適 iPhone X 的壁紙 App ﹣ Cube X

Cube X 是我在體驗了一圈壁紙 App 之後，我發現，根本沒有真正合適手機壁紙的 App。每一個壁紙 App 或多或少都有如下這些缺點：
* 分辨率不滿足要求；
* 奇怪的裁剪方案；
* 預覽顯示並非合適手機屏幕尺寸。

我們要選擇一張壁紙，首先需要一點就是，這個壁紙是基於當前手機比例進行預覽的。而不是一個一個正方形格子……這樣無法選擇啊！

因此 Cube X 使用了獨特的 1:2 比例，以滿足 iPhone X 手機的預覽尺寸。讓每個用戶在瀏覽畫廊的時候，就可以做出一種選擇，我到底要不要用這張壁紙。



不但如此，Cube X 更加關心你怎麼預覽這張壁紙。在 Cube X，一共有 3 種屏幕預覽模式。還有視覺效果的預覽，以滿足你會在壁紙設定過程中會設定為「視差效果」。

同時在壁紙屬性面板還提供了基於 unsplash.com 的點讚接口。不能忘記了，Cube X 最為重要的功能就是，提供 RAW 原片下載。一張 10m 的壁紙的威力，不是一張 300kb 的壁紙可以比較的。

基於 unsplash.com api 和自定義接口的 stocksnap.io 兩款 CC0 的素材圖庫，無論你是搜索圖片做 PPT 還是搜索圖片來做壁紙，甚至於搜索圖片來打印傳遞，通通沒問題！

這麼完美的壁紙 App，就要現在下載。

優點
  • 符合手機顯示尺寸的預覽比例
  • RAW 原片下載
  • 多引擎搜索接口
缺點
  • unsplash.com API 速度偏慢
  • 偶然的可能錯誤
  • 依然正在更新中……
